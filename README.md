Este repositório foi criado no âmbito da UFCD 10805 - Programação em Python.

O objetivo é guardar e organizar os exercícios realizados durante a formação, aplicando também os conceitos de Git, GitHub, commits, branches, pull requests e documentação.

O projeto contém ficheiros Python desenvolvidos ao longo dos módulos da formação.

Exemplos de conteúdos incluídos:

- Exercícios sobre tipos de dados;
- Exercícios sobre estruturas de dados;
- Exercícios sobre conceitos genéricos de programação;
- Atividades relacionadas com Git e GitHub.
